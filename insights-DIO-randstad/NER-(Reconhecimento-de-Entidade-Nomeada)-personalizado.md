## **NER (Reconhecimento de Entidade Nomeada)**


Com o NER personalizado, é possível desenvolver modelos de IA especializados capazes de identificar e extrair entidades específicas em textos não estruturados, como contratos jurídicos ou relatórios financeiros. Durante a criação de um projeto desse tipo, o usuário pode rotular dados de maneira iterativa, realizar o treinamento e a validação do modelo, além de aplicar melhorias contínuas para elevar sua precisão antes da implantação.

A qualidade dos dados rotulados desempenha um papel fundamental, pois influencia diretamente a confiabilidade e a exatidão dos resultados.

Para tornar o processo de configuração mais simples e intuitivo, o serviço disponibiliza um portal web dedicado, acessível por meio do Language Studio. Para iniciar rapidamente, basta seguir as orientações descritas no guia de início rápido.


- Ciclo de vida de desenvolvimento do projeto

Etapas principais para NER personalizado:

1 Definir o esquema → conhecer os dados e identificar as entidades a serem extraídas, evitando ambiguidades.

2. Rotular os dados → realizar anotações de forma precisa, consistente e completa, garantindo qualidade no treinamento.

3. Treinar o modelo → permitir que o modelo aprenda a partir dos dados rotulados.

4. Avaliar o desempenho → analisar métricas e resultados para identificar pontos de melhoria.

5. Implantar o modelo → disponibilizar o modelo treinado via API para uso em aplicações reais.

6. Extrair entidades → aplicar o modelo personalizado em textos para obter as entidades definidas.
